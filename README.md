# 📚 TutorHub

🌐 Live Site: https://mediqueue-fawn.vercel.app

TutorHub is an online tutoring platform where students can find tutors, book sessions, and manage learning schedules easily.

## ✨ Features

➤ 🔍 Search tutors by tutor name  
➤ 📅 Filter tutors using registration start and end dates  
➤ 🔐 Secure private routes with JWT authentication  
➤ 🧑‍🏫 Add, manage, and delete tutoring sessions  
➤ 📖 Book tutoring sessions with real-time slot updates  
➤ ❌ Cancel booked sessions anytime  
➤ 🌙 Responsive modern UI with dark/light theme support  
➤ ⚡ Fast performance using Next.js and Express.js  

## 🛠️ Technologies Used

- Next.js
- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Vercel
